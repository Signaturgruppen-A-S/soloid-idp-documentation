# Technical documentation for SoloID Identity Provider

## Easy certified authentication
SoloID identity provider is an identity provider that is certified to provide an NSIS substantial level of authentication. This is the same level of authentication as the most common MitID level of authentication

It provides an easy and seamless authentication and registration experience for users.

This provider can be used as a MitID drop in replacement for many use cases. Compared to MitID it is a more cost effective solution that also provides a faster & easier authentication experience

## MitID and Passkeys
SoloID identity provider provides a range of authentication methods. For a nice and seamless authentication experience MitID can be combined with passkeys, allowing users to use passkeys for frequent authentications providing an easier and more phising resitant authentication. 
MitID becomes a secondary option, providing access in circumstances where users do not have a passkey available. The combination with MitID allows users to easily manage their passkeys and reduces the typical hassle with losing access to passkeys.

## Enterprise
This identity provider can be used in many enterprise settings for improving authentication, registration and onboarding enterprise users. It can also be used to authenticate a user to various services, such as microsoft online. With deep integrations to Microsoft Entra, Active directory as well as SCIM provisioning support, we offer an easy way to authenticate users.
